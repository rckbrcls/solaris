# Getting Started

## Requirements

- macOS with Xcode installed.
- iOS simulator or physical iOS device.

## Open the Project

Open `solaris.xcworkspace` when CocoaPods/workspace integration is needed, or `solaris.xcodeproj` when working only with the Xcode project.

## Running and Testing

Use Xcode schemes for local development, previews, simulator runs, and tests. No command-line build or test command was verified during this documentation pass.
